Cara Kerja

Prediksi Sentimen

Melakukan prediksi sentimen terhadap tweet publik terkait Corona / Covid-19.

  • Klik menu Prediksi Sentimen
  • Masukkan keyword yang berkaitan dengan Corona pada textbox yang tersedia
  • Klik tombol Kirim untuk memulai prediksi
  • Sistem akan mengambil data twitter sesuai dengan keyword yang telah dimasukkan
  • Server akan melakukan proses pembersihan data hingga klasifikasi
  • Menampilkan hasil prediksi

Cara Kerja

Komparasi Model

Membandingkan performa kombinasi kernel pada algoritma Support Vector Machine dan Term Frequency terhadap analisis sentimen.

Skema model:

  1. TF-IDF & Kernel Polynomial
  2. TF-IDF & Kernel Radial Basis Function
  3. TF-RF & Kernel Polynomial
  4. TF-RF & Kernel Radial Basis Function
  • Klik menu Komparasi Model
  • Masukkan file dengan ekstensi csv
  • Pastikan terdapat 2 kolom yaitu kolom Teks dan Label
  • Kolom Teks berisikan data yang akan diprediksi
  • Kolom Label berisikan kelas dari kolom Teks, dimana terdapat kelas Positif, Negatif, dan Netral
  • Klik tombol untuk memulai proses komparasi
  • Menampilkan hasil komparasi model
